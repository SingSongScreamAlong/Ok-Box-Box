# Ok Box Box - Beta Tester Guide

Welcome to the Ok Box Box beta! This guide will help you get set up with the Driver Tier.

---

## What You'll Need

- **Windows 10/11** PC
- **iRacing** subscription and installed
- **Email address** for account creation

---

## Step 1: Create Your Account

1. Go to **[app.okboxbox.com](https://app.okboxbox.com)** (or the URL provided)
2. Click **Sign Up**
3. Enter your email and create a password
4. Check your email for verification link
5. Click the link to verify your account

---

## Step 2: Install the Relay

The relay is a small app that reads telemetry from iRacing and sends it to Ok Box Box.

### Download
- **[Download Ok Box Box Relay](https://okboxbox.com/download)** (Windows only)

### Install
1. Run the installer (`Ok Box Box Relay 1.0.0-alpha.exe`)
2. Follow the installation prompts
3. The relay will appear in your system tray

### First Run
1. Launch the relay from your Start menu or system tray
2. Sign in with your Ok Box Box account
3. The relay will show "Waiting for iRacing..."

---

## Step 3: Connect to iRacing

1. **Start iRacing** and enter a session (practice, race, etc.)
2. The relay will automatically detect iRacing
3. Status will change to "Connected to iRacing"
4. Open **[app.okboxbox.com/driver/cockpit](https://app.okboxbox.com/driver/cockpit)** to see live telemetry

---

## Known Limitations (Alpha)

- Relay only works on Windows
- Must have iRacing running for live telemetry
- Some features require iRacing account linking (coming soon)
- Voice features require microphone permission

---

## FAQ

### The relay says "Waiting for iRacing" but iRacing is running
- Make sure you're in an active session (not just the UI)
- Try restarting the relay
- Check that iRacing's telemetry is enabled in settings

### I don't see my iRating/licenses
- Your iRacing account may not be linked yet
- Go to Settings and connect your iRacing account

### The cockpit shows no data
- Ensure the relay is running and connected
- Check that you're logged into the same account in both
- Refresh the browser page

### Can I use this on Mac/Linux?
- Not yet. Windows only for now due to iRacing requirements.
